How to Control Windows Input Volume and Mute with MIDI

Map a fader/knob to input level and a button to input mute in Windows.

This workflow lets you control microphone or line-in input endpoints directly from your MIDI controller. You can target the default Windows input, or select a specific input endpoint.

  1. Add and enable your MIDI device in MIDI Command Studio.
  2. Click Learn Fader/Knob and move the control you want to use.
  3. In the mapping editor, set controls to Volume - Input (Default).
  4. Optional: to select inputs other than default, in Advanced settings, enable show specific windows input devices. This will allow the controls list to show all windows inputs.
  5. Save and move the control to verify input level changes.
  6. Choose one mute setup method below, then save and test mute/unmute.

Two Ways to Map Mute

  • Paired mute (recommended with input fader): in the same fader mapping, enable Add mute control, click Learn, and press the mute button you want to pair.
  • Standalone mute mapping: learn a button/pad, open its mapping editor, click Use as mute control..., then select your input target in the mute dialog.
