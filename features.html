Features

Core capabilities, fast setup, and precise control for Windows.

MIDI Command Studio turns a MIDI controller into a dedicated control surface for everyday Windows work. Build mappings quickly, target specific apps, and switch presets without breaking flow.

Mapping and Control

Dedicated learn buttons for each control type let you assign mappings quickly and accurately.

Presets and Workflow

Create presets and switch quickly during a session.

Switch presets from the app or directly from your controller.

Autosave

Presets and device settings are saved automatically, so you don't need to worry about losing any changes.

LED Feedback

Light up your controller's mapped buttons with always-on, momentary, or toggle options (Device must support LED Feedback).

App Targeting

Send mappings to a specific app or keep them global across Windows.

Volume Control

Control system volume or a specific application's volume with a fader or knob.

Shortcut Palette

Pick shortcuts from a built-in library for different programs (auto-fills the description), or capture your own.

MIDI Thru

Pass unmapped MIDI controls through to other apps using a virtual MIDI port (requires 3rd party virtual MIDI port).

Behringer X-Touch One

Device-specific support for the X-Touch One.

On the Behringer X-Touch One, MIDI Command Studio supports the motorized fader and LCD/scribble strip. The display can show the current preset, which is useful when switching presets from the controller. The fader can automatically move to match the target app’s volume when you switch to a preset where the fader is mapped to volume. This is useful when using the fader for volume control across different apps. You can effectively control multiple app volumes and the system volume with one fader.
