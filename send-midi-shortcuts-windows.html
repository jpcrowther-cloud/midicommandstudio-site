How to Send Keyboard Shortcuts with a MIDI Controller on Windows

Map buttons, pads, knobs, and faders to shortcuts for any Windows app.

Add your MIDI device, learn a control, choose a shortcut, optionally target a specific app, then save the mapping.

  1. Open MIDI Command Studio and add your device in the Devices panel.
  2. Click the appropriate Learn button and move or press the control (editor will open automatically).
  3. In the editor, click a shortcut field and choose a shortcut from the Shortcut Palette or capture your own shortcut.
  4. Set App Targeting if needed.
  5. Click Save and test the control.

Use global mappings for system-wide hotkeys, or app-specific mappings so controls only affect the app you are working in.

Troubleshooting

  • If Learn does not detect input, confirm the selected device is green/connected.
  • If the target app is not showing in dropdown, make sure the target app is running.
  • If controls behave unexpectedly, using the built-in Log window helps identify what control type your MIDI device is using.

Full details are in the Help guide.
