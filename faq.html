FAQ

Quick answers to common setup and licensing questions.

Is there a free version?

Yes. MIDI Command Studio has a Free version and a Full version. See Free vs Full for the feature limits.

Can I use more than one active device?

Free supports one active device at a time. Full supports multiple active devices.

Can I control app volume as well as system volume?

Yes. You can map controls to system volume or target a specific app volume.

Why does Windows show a warning when I run the installer?

Windows may show SmartScreen or security warnings for newer unsigned installers. See the notes on the Download page.

Where do I get support?

Support is handled in-app via Settings > Feedback. This keeps reports organized and can include diagnostics when needed.

What data is sent by the app?

The app sends data for licensing actions and optional feedback/crash reporting only. See Data and Privacy for details.

Does it work with all MIDI controllers?

It works with most controllers that expose standard MIDI controls on Windows. Some devices may require specific mode settings.

Where is the full documentation?

Use the Help page for setup and detailed guides.
