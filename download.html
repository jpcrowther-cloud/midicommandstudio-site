Windows security note: New apps sometimes trigger a warning on first run. If you see “Windows protected your PC,” click More info → Run anyway. On some Windows 11 systems, Smart App Control can block unsigned apps entirely, so this beta might not launch if SAC is enabled.

System requirements: Windows 10 or 11, a MIDI controller or MIDI keyboard, and a standard user account (no admin rights required after install).

Quick install: 1) Download the installer. 2) Run it and follow the prompts. 3) Launch MIDI Command Studio from the Start menu and connect your controller.
