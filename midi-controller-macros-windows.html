MIDI Controller Macros for Windows Apps

Run multi-step actions from one MIDI button.

Learn a button, open Macro in the editor, add steps, and save. A single press can launch apps, open files, send shortcuts, type text, click the mouse, and add delays.

  1. Learn a button or pad in Mappings.
  2. In the editor, choose Macro.
  3. Add steps.
  4. Add a description.
  5. Save and test.

You can add any Macro created to a favourites list for saving and easy access.

Example Macro Ideas

  • Open app, wait, then send a startup shortcut sequence.
  • Switch scene in streaming software and trigger a hotkey.
  • Launch a utility and open a specific file path.
  • Run repetitive editing actions from one button press.
