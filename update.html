What's new in 1.2.0

  • Added one-button preset cycling for faster controller workflows.
  • Improved encoder-based volume control for System, App, and Input targets.
  • Improved Mackie/MCU feedback support and setup flow for compatible devices.
  • Added overlay display for triggered mappings and actions.
  • Improved preset-switch conflict handling across saved presets for the same device.
