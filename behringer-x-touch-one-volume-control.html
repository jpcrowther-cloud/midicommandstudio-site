Behringer X-Touch One: Control Multiple Windows Volumes with One Fader

Use one motorized fader to control different targets by switching presets.

X-Touch One MIDI Mode Quick Setup

  1. Hold the STOP button and press the rotary knob.
  2. Use the rotary knob to select MIDI.
  3. Press the rotary knob multiple times until the LCD screen has no text.

One-Fader Volume Setup

Create separate presets for each volume target, map the X-Touch One fader in each preset, and switch presets from the controller.

  1. Add your X-Touch One in MIDI Command Studio.
  2. Create a preset for each volume target you want (for example: System, Browser, Spotify, VLC).
  3. In each preset, map the motorized fader to the volume you'd like to control.
  4. Enable motorized fader feedback in Settings for the X-Touch One.
  5. Switch presets from the controller to change what the fader controls.

With motorized feedback enabled, the fader can move to the target level when you switch presets so one physical fader can manage multiple volume targets.

Important Notes

  • For App Target volume, the app must be running and producing audio.
  • Preset switching from the controller is the key to one-fader multi-target volume control.
  • Preset switching can be mapped via the Settings/Preset window.
  • The X-Touch One LCD can show preset names to make target switching clearer.
  • This method also works with other devices, but motorised feedback and LCD feedback are not currently supported (more devices will be added over time).
